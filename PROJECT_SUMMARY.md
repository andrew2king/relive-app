## 🚀 快速启动指南

### 1. 环境准备
```bash
# 克隆项目
git clone <repository-url>
cd relive-app

# 复制环境配置
cp .env.example .env
# 编辑.env文件，配置必要的API密钥
```

### 2. 一键启动
```bash
# 运行自动化启动脚本
./scripts/start-dev.sh
```

### 3. 手动启动
```bash
# 安装依赖
npm run install:all

# 启动数据库
docker-compose up -d postgres redis

# 运行数据库迁移
cd backend && npx prisma migrate dev && cd ..

# 启动开发服务
npm run dev
```

### 4. 访问应用
- **前端应用**: http://localhost:3000
- **API文档**: http://localhost:8000/api-docs
- **数据库管理**: http://localhost:5432
